# **App Name**: VYA Logistics

## Core Features:

- Unified App Interface: Single app with modes for senders and travelers.
- Dynamic Pricing: Determine cost to send packages, dependent on size and distance using fixed, pre-arranged prices.
- Modal Expansion: Support for cars, motorcycles, and intermunicipal buses.
- Dynamic Meeting Points: Algorithm suggests secure meeting points along traveler routes.
- Real-time Behavioral Tracking: Visual alerts to ensure recipient readiness at meeting points. A tool confirms traveler is at meeting location to speed up delivery process.
- Fiscal Check-in: Mandatory upload of electronic invoices for legal compliance.
- VYA Wallet: In-app wallet for instant payouts and transactions.

## Style Guidelines:

- Primary color: Deep purple (#702963), inspired by travel and logistics
- Background color: Light purple (#F2E7F2)
- Accent color: Blue (#296370) to represent trust and reliability in transit
- Headline font: 'Poppins' sans-serif font for contemporary, geometric style
- Body font: 'PT Sans' sans-serif font to make longer text easy to scan.
- Use of clear, minimalist icons for modes of transport (car, moto, bus) and package sizes (P, M, G).
- Clean and intuitive design for sender/traveler mode switching.
- Smooth transitions and loading animations to enhance user experience during package tracking and mode switching.
